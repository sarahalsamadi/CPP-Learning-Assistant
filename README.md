# CPP Learning Assistant

An educational tool built with C++ to help beginners master the fundamentals of programming through a hands-on CLI experience.

## ✨ Features
* [cite_start]**Secure Login System:** Simple authentication to start the learning journey.
* [cite_start]**Comprehensive Tutorials:** Covers Introduction, Variables, Loops, Functions, and Arrays.
* [cite_start]**Code Examples:** Real snippets to demonstrate how logic works in practice.
* [cite_start]**Interactive Quiz:** A built-in test with automated scoring to evaluate your progress.

## 🛠️ Built With
* **Language:** C++
* **Environment:** Code::Blocks
* [cite_start]**Key Concepts:** Control Structures, Switch-Case logic, Procedural Programming.

## 🚀 How to Use
1. Clone the repository.
2. Open `Project.cbp` using Code::Blocks.
3. Compile and Run `main.cpp`.
4. Follow the on-screen instructions to navigate through the topics.
